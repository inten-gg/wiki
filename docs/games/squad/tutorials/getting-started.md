# Getting Started with Squad

TL;DR: What you need and how to join the server.

## Prerequisites
- Steam account
- Install Squad via Steam

## Steps
1. Join our Discord and request access.
2. Launch Squad via Steam.
3. Open the server browser and connect to our server or use direct connect: `connect ip:port`
4. Select a role and follow the squad leader.

## Tips
- Use proximity voice and stick with your squad.
